# react-cesium
一个在react中加载cesium的demo


不同分支是项目不同时期的发展形态
* `demo/simple` 分支的代码是一个最最最简单在react内显示出一个cesium控件的代码

# usage

本项目是使用`create-react-app` 脚手架创建的，按通常的项目运行即可
大体流程如下：
+ 先克隆到本地
+ 根据上面的说明切换到需要的分支
+ 运行`yarn` 命令安装依赖
+ 运行`yarn start` 启动开发服务器
